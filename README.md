# Puff - Morning Breathwork Companion

**Tagline:** Breathe. Begin. Bloom.

A simple, beautiful breathing exercise app to help you start your day with intention and calm.

![Puff Mascot](https://via.placeholder.com/120x60/e0f2fe/7dd3fc?text=Puff)

## Features

- **3 Proven Breathing Patterns:**
  - 4-7-8 Technique (calms anxiety)
  - Box Breathing (focus & clarity)
  - Coherent Breathing (balance & harmony)

- **Session History Tracking** - See your progress over time
- **Offline Access** - Practice anywhere, anytime
- **Free Forever** - No subscriptions, no ads

## Getting Started

```bash
npm install
npm run dev
```
